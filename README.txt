Homefiles
===========

the folders containing dashes are interesting

simple dash: -
--------------

like 'sh-shellfiles' 

    - link folder contents directly under ~/.
    - link folder contents directly under ~/sh

dot dirs: -dot-
---------------
like 'sh-dot-mydir'

    - link folder contents  under ~/.mydir
    - link folder contents  under ~/sh

home dirs: -home-
---------------
like 'sh-home-mydir'

    - link folder contents  under ~/mydir
    - link folder contents  under ~/sh
